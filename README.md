ขั้นตอนการทำงาน
1.pnpm install vue,nuxt
2.ทำ component
3.เอาcomponent ใส่หน้าเว็บ
4.ทำ api โดยสร้าง table ชื่อ counter_logs
5.รัน local โดยใช้คำสั่ง pnpm dev และ รัน server.js โดยใช้คำสั่ง node server.js พร้อมกัน local คือ frontend และ server คือ api
6.หลังจากนั้น เข้าไปในหน้าเว็บ และลอง save และลองเข้าไปเช็คใน database (table counter_logs)
