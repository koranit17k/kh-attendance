## project สำหรับการบันทึกและนำข้อมูลไปใช้
เป็นโปรเจคที่เรียนรู้การบันทุกข้อมูลจากหน้า frontend บันทึกข้อมูลเข้าไปยัง database และ นำข้อมูลจาก database ไปใช้ต่อ
> git: https://github.com/koranit17k/kh-attendance.git

### ขั้นตอนการทำงาน
1. pnpm install vue,nuxt
2. ทำ component
3. เอาcomponent ใส่หน้าเว็บ
4. ทำ Database โดยสร้าง table ชื่อ counter
5. สร้างโฟลเดอร์ server เพื่อให้นำมาใช้เป็น api
    - api เพื่อเป็นตัวรับส่งข้อมูล ที่ได้จากการนับข้อมูลในเว็บ
    - middleware เพื่อเป็นการนับ session,cookie ในเว็บ
    - utils เพื่อเป็นการเชื่อมกับ database
6. รัน local โดยใช้คำสั่ง pnpm dev 
7. หลังจากนั้น เข้าไปในหน้าเว็บ และเข้าไปเช็คใน database (table counter)
8. สร้าง calendar เพื่อเช็ควันที่พนักงานมาทำงาน 
9. สร้าง attend-sum เพื่อเป็น kpi/card เพื่อเช็คจำนวนวันที่พนักงานมาทำงาน เต็มวัน, ครึ่งวัน, ขาด
10. สร้าง call-report เพื่อเป็นตัวเรียกใช้ report โดยใช้ api 


Note*
- อย่าลืม ตั้งค่า config ใน nuxt.config.ts, package.json, tsconfig.json
