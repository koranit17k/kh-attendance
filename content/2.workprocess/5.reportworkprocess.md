> **Git:** [kxreport](https://github.com/swasin185/kxreport.git)


### 🛠️ 1. การเตรียมระบบ (Environment Setup)
*   **Clone Project:** ทำการ Clone หรือ Fork Repository จาก GitHub
*   **สิ่งที่ต้องติดตั้ง:**
    - Node.js & pnpm
    - Tomcat 10
    - Java (JDK)
*   **โครงสร้างโฟลเดอร์:** ต้องมีโฟลเดอร์ `kh-group` อยู่ใน Root หลังจาก script/build.sh

---

### 📄 2. การจัดการรายงาน (Report Development)
*   **Compile Report:** เมื่อสร้างรายงานด้วยไฟล์ `.jrxml` เสร็จแล้ว ต้องทำการ Compile ให้เป็นไฟล์ `.jasper`

---

### ⚙️ 3. การติดตั้งและรันระบบ (Build & Run)
*   **ติดตั้งโปรแกรม:** รัน `script/install.sh` เพื่อติดตั้งโปรแกรมที่ใช้แสดงผล Report
*   **สร้างโครงสร้าง:** รัน `script/build.sh` เพื่อเตรียมระบบให้พร้อมทำงาน
*   **ทดสอบการใช้ทรัพยากร:** รัน `script/mem.sh` หรือ `script/stress.sh` เพื่อทดสอบการใช้งาน RAM และความเสถียรของระบบ
*   **Script คัดลอก:** รัน `script/copyreport.sh` เพื่อคัดลอกไฟล์ `.jasper` ไปยังโฟลเดอร์ `payroll` โดยอัตโนมัติ
*   **การจัดเก็บ:** ไฟล์รายงานจะถูกเก็บไว้ที่ Path: `/khgroup/report/payroll/`

---

### 🔍 4. การทดสอบและใช้งาน (Test & Usage)
*   **ทดสอบ API หลัก:** สามารถเข้าทดสอบผ่าน [http://localhost:8080/kxreport/](http://localhost:8080/kxreport/)
*   **ตัวอย่างการเรียกรายงาน (PDF):**
    - เรียกผ่าน URL โดยส่ง Parameter เช่น `report`, `db`, `startDate`, `endDate`, `comCode`
    - **ตัวอย่าง:** [http://localhost:8080/kxreport/getPDF?report=A02&db=payroll&startDate=2025-12-01&endDate=2025-12-31&comCode=01](http://localhost:8080/kxreport/getPDF?report=A02&db=payroll&startDate=2025-12-01&endDate=2025-12-31&comCode=01)
